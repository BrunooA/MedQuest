# 🏥 MedQuest — Aplicativo para Monitoramento de Saúde

O **MedQuest** é um aplicativo mobile focado em centralizar e facilitar o controle de saúde dos cidadãos. O projeto nasceu no primeiro bimestre como um protótipo interativo desenvolvido no Figma e, no segundo bimestre, foi transformado em uma aplicação funcional utilizando **Flutter** e **Dart**. 

Este projeto foi desenvolvido como parte das matérias de UI/UX Design e Desenvolvimento Mobile na UniSales (Vitória - ES).

---

## 🎯 Desafio de UI/UX & Acessibilidade
O principal objetivo do design foi criar uma interface clara, moderna e altamente intuitiva, capaz de atender com facilidade diferentes perfis de usuários, desde jovens conectados até idosos e pacientes crônicos que necessitam de maior clareza visual e jornadas simplificadas.

### Funcionalidades Projetadas:
* **Acompanhamento de Saúde:** Gráficos de evolução semanal e check-ins diários de bem-estar.
* **Agendamentos:** Organização e controle de consultas e exames médicos.
* **Medicamentos:** Lembretes intuitivos para controle de medicação.
* **Telemedicina:** Interface preparada para suporte a chat e videochamadas com profissionais da saúde.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **[Figma](https://www.figma.com/):** Prototipação de alta fidelidade, definição da paleta de cores e fluxo de telas.
* **[Flutter](https://flutter.dev/) & [Dart](https://dart.dev/):** Implementação do código-fonte, componentização em Widgets e navegação fluida (`Navigator`).
* **Google Forms:** Validação de usabilidade e testes com usuários reais para colher feedbacks da interface.
